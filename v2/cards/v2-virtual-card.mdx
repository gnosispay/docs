To create a virtual card, you need:
- Verified phone number

## Card Activation

Virtual cards are activated immediately after creation and can be used for online purchases right away.

To create card for user, call [`POST /cards/virtual`](/api-reference/cards/create-virtual-card) along with cardName. Once successful, you will get a CardID.

```bash
curl --request POST \
  --url https://gp-auth-module.sandbox.gnosispay.in/cards/virtual \
  --header 'Content-Type: application/json' \
  --data '{
  "cardName": "<string>"
}'
```
